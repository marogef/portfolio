My Projects

  • IP-finder

    Used an IP lookup API to quickly and simply integrate IP geo-location into the website to find hostname, network, city, latitude, postal code, and route. This is for businesses who want to identify visitor's geographical location and data. It was built to help them identify this information. It was implemented with HTML, CSS, AJAX, JavaScript, and the geolocation services API.

    Live Preview Access GitHub

  • Bestbuy

    Searching for electronic products or get pointers on where to go in order to get the best electronic deals. It uses the Best Buy API in order to make the search. From there you can add an item and store it as a favorite in your list that is very useful for all users. This was built for shoppers who like to buy from Bestbuy. It is used in order to search for any item in Best Buy and help them with their experience. It was implemented using HTML5, CSS, JavaScript, Jquery, AJAX, and the Bestbuy API.

    Live Preview Access GitHub

  • Yummly

    Full stack and react - Using Yummly.com API to do recipe search for food. Search for any food item of the user's choice. You can add an item from the search results with a click of a favorites button. It is built for anyone who wants to learn and add more recipes to their cooking. It was built to help people who want to cook with more options. This was implemented using HTML5, CSS, Javascript, JQuery, AJAX, React, and the Yummly API.

    Live Preview Access GitHub
